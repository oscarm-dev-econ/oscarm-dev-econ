# oscarm-dev-econ — GitHub Pages site

Live site: https://oscarm-dev-econ.github.io/oscarm-dev-econ/

This repo hosts a simple Markdown site rendered by GitHub Pages (Jekyll + a built-in theme).

## Structure
├── _config.yml # Site settings (theme, title)
├── index.md # Home page
├── about.md # About page
└── assets/
└── img/ # Images go here
